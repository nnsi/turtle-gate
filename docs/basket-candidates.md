- 全セクターOC（現行）: 0.43

- A+B群PTS（楽観的）: 0.59×0.65 + 0.41×0.43 = **0.56**

OC/CC比率の改善は 0.43 → 0.51-0.56。「0.65-0.70に改善」という提言書の想定は全セクターPTS前提であり、現実的には 0.51-0.56 が期待値。
